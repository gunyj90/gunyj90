# 📊 임신 준비 \~ 임신초기 (여성 / 남성 통합 생활·검사 가이드)

---

# ✔️ 핵심 정리

👉 이 가이드는 “정보”가 아니라  
**생활 + 검사 + 위험관리까지 포함된 실행 기준표**

- 임신 전: 몸 상태 만들기 + 검사
- 시도기: 타이밍 + 컨디션
- 초기: 안정 + 검사 타이밍
