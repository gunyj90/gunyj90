# ✔️ 핵심 요약 (리스크 중심)

- “먹는 것보다 피하는 것이 더 중요할 때가 있다”
- 초기 임신은 **환경·습관 영향이 매우 큼**
- 남성의 흡연/생활습관도 실제로 영향 있음
- 가장 큰 리스크 3가지:
  👉 술 / 흡연 / 과로
